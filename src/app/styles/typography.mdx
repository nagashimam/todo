# タイポグラフィー

## フォントファミリー

大きく以下の 2 種類に分けています。

1. 読ませるためのフォント(段落に使う)
2. 見せるためのフォント(段落以外に使う)

前者は明朝体、後者はゴシック体です。

表示速度を優先するために、Web フォントではなく OS 標準フォントを使っています。

## フォントカラー

基本的に--primary を使います(背景が--primary の場合は--secondary)。.important を指定した箇所のみ--accent になります。

色だけで意味を伝えるのは好ましくないので、.important を指定した箇所は font-weight と font-size も合わせて変えています。[色](/docs/styles-color--color)
